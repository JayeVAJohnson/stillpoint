What's inside

Five small tools.
One quiet page.

FOCUS

Pomodoro timer

Configurable focus, break, and long-break lengths. Tracks sessions and flashes the tab title when one ends.

TASKS

To-do list

Add with Enter, edit inline, import or export as .txt or .csv. Clear done tasks in one click.

NOTES

Notepad

A plain scratch space that auto-saves as you type and persists across sessions.

STREAKS

Habits

A 7-day dot grid per habit. Click any dot to toggle it — today's dot gets a ring.

SCHEDULE

Day planner

Time blocks or a simple ordered list, your choice. Resets fresh each day.

Make it yours

Six themes, plus your own photo

Pick a built-in palette or upload a personal background — stored locally, never uploaded anywhere.

Setup

Two ways to run it

Same files, same features. Pick whichever sounds easier.

01
Download this repo as a ZIP from GitHub and unzip it somewhere permanent — don't move or delete the folder afterward.
02
Open chrome://extensions — works for Chrome, Brave, Arc, Opera, Vivaldi, and Avast Secure Browser. Edge uses edge://extensions. Firefox is a little different (see the README).
03
Turn on Developer mode in the top right, then click Load unpacked.
04
Select the unzipped folder. Open a new tab — that's it.
01
Download this repo as a ZIP from GitHub and unzip it anywhere on your computer.
02
Double-click newtab.html — it opens in your default browser like any normal page. No developer mode, no permissions screen.
03
Bookmark it, or set it as your browser's homepage in Settings → "On startup."
04
Everything still saves locally and is still there next time you open it — it just won't auto-replace every new tab.

Everything stored locally — your browser, your machine, nowhere else.
